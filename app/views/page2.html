Dessiner des cercles avec P5 et Processing est facile.
On peut s'appuyer notamment sur les primitives "ellipse" et "arc" : 


        ellipse(x,y,w,[h])
        arc(x,y,w,h,angleStart,angleStop,[mode])

  • Documentation P5 : http://p5js.org/reference/
  • Documentation Processing : https://processing.org/reference/
