Précalculer et stocker les coordonnées de notre cercle offre certains avantages.
On peut exploiter les données précalculées pour générer un effet de traîne.
Pour ce faire, on va réutiliser le sketch précédent, mais pour chaque point "dessiné", nous allons afficher les 5 points précédents avec un effet de dégradé. 
// boucle de tracé des 5 points "suiveurs" à placer après le tracé du point principal
for (var j = i-1 , k=0; j >= 0 && k<=5 ; j -= 1, k--) {
    col += 5;  // effet de dégradé pour les points "suiveurs"
    stroke(col);
    point(data[j].x, data[j].y); // tracé des points "suiveurs"
}
