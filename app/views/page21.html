Utilisation de P5 avec Angular

Cette page est destinée aux développeurs Javascript plus avancés.

L'utilisation de P5 avec Angular nécessite quelques ajustements au niveau des sketchs.

Dans l'application que vous êtes en train de consulter, les sketchs sont rattachés à un contrôleur Angular, sous la forme de "factories" Angular (une "factory" par sketch).

Cette architecture nécessite de modifier chaque appel de fonction angular pour le préfixer avec le code que vous avez défini en entrée de la fonction "factory" encapsulant le sketch (par exemple "p").

Vous trouverez un exemple de code dans le source "P5 et Angular".
