Un sketch P5 ou Processing contient à minima les 2 fonctions setup() et draw() 


    // avec P5.js                    
    function setup() {
    }
    function draw() {
    }
        

    // avec Processing                    
    void setup() {
    }
    void draw() {
    }

La foncion setup() est exécutée une seule fois, tandis que la fonction draw() est exécutée en boucle, à un rythme théorique de 60 FPS (Frame Per Second).

Si vous connaissez un peu Arduino, vous remarquerez la grande similitude avec les fonctions setup() et loop() d'Arduino.
